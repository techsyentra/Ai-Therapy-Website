# Psychotherapy Now - AI-Supported Psychotherapy Platform

## Introduction

Welcome to the Psychotherapy Now website project! This platform provides a professional, AI-assisted interface designed specifically for Licensed Psychotherapists. It aims to increase patient awareness and facilitate behavioral change through structured, therapist-supervised dialogue.

## Objectives

The primary goal is to provide licensed professionals with a secure, HIPAA-compliant environment where they can conduct supervised AI-assisted therapy sessions. The platform empowers therapists by functioning as a controlled tool that leverages established therapeutic techniques (such as CBT, NLP, Gestalt, and Ericksonian Therapy).

## Core Features

1. **Supervised AI-Assisted Sessions**
   - Conduct structured 15–45 minute sessions utilizing AI in the therapeutic process.
   - The therapist maintains full real-time control, session authority, and clinical responsibility over the AI's interactions.
2. **Secure & HIPAA-Compliant Environment**
   - Encrypted audio rooms with exclusive clinician access.
   - Strict privacy controls with no retaining of identifying patient data.
3. **Professional Support Framework**
   - AI programming is integrated with multiple proven theoretical frameworks and over 50 years of training experience.
4. **Therapist Onboarding & Verification**
   - Secure verification of specific state and professional licenses before allowing platform usage.
   - Clear onboarding instructions detailing platform features like Voice Control and Whisper Control techniques (to direct the AI dynamically).
5. **Flexible Pricing System**
   - Transparent rates tailored for clinical usage (e.g., $25 for 100 session minutes on a Starter Package).
6. **Responsive & Modern Interface**
   - Clean landing platform built with semantic HTML and custom CSS.
   - Sections detailing platform guidelines, features, therapist onboarding timelines, and secure sign-up/login processes.

## Technologies Used

- **HTML5**: Semantic and accessible structuring of the platform's core pages (including `index.html`, `signup.html`, `login.html`, etc.).
- **CSS3**: Completely responsive, premium dark-themed aesthetic with custom utility classes, modern flexbox/grid layouts, and interactive animations.
- **JavaScript (Vanilla)**: Handles essential UI interactions, responsive navigation menus, and dynamic front-end elements.
- **Ionicons**: For lightweight, scalable vector icons throughout the interface.

## Project Structure

- `index.html`: The main landing and informational page detailing features, the clinical framework, onboarding, and pricing packages.
- `login.html` & `signup.html`: Fast authentication portals for the psychotherapists.
- `assets/css/style.css`: The core stylesheet responsible for all styling, variables, layout systems, and responsive behavior.
- `assets/images/`: Organized graphical assets and banners used across the pages.

## Getting Started

To view the marketing and interface platform locally:
1. Clone or download this repository to your local machine.
2. Open the directory folder.
3. Launch the `index.html` file in any modern web browser (Google Chrome, Firefox, Safari, Edge).
4. Navigate across different sections to experience the fully responsive, therapist-focused application interface.

## Conclusion

Psychotherapy Now bridges the gap between traditional professional counseling and modern AI technology. It provides clinical therapists with an advanced technological framework that adheres strictly to established clinical compliance, security standards, and ethical boundaries.
